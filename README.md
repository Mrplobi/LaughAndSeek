- https://freesound.org/
- Dotween V2 (https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676)
- Meta Core SDK (https://assetstore.unity.com/packages/tools/integration/meta-xr-core-sdk-269169)
